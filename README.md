
# Java Swing Quiz Application

## Description
A desktop-based Quiz Application built using Java Swing. Users can answer multiple-choice questions and get their score at the end. Demonstrates GUI design, event handling, and core Java concepts.

## Features
- 5 multiple-choice questions
- Score calculation and result display
- User-friendly GUI with Swing components
- Event-driven button handling
- Dynamic question and option update

## Skills Used
- Core Java (loops, arrays, methods)
- Object-Oriented Programming (OOP)
- Java Swing GUI development
- Event handling with ActionListener

## How to Run
1. Open project in any Java IDE (Eclipse, IntelliJ, NetBeans).
2. Run `quiz.java` file.
3. Alternatively, compile and run from terminal:
   ```bash
   javac quizapp/quiz.java
   java quizapp.quiz
